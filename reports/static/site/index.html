Push data

DATA='{
    "key": "AMJ1IG2N4UEJUR2K",
    "rows": {
        "label": "users",
        "count": 10
    }
}'
curl -XPUT --data "$DATA" http://reports/api/v1/report/1/

Visualize data

DATA='{
    "key": "AMJ1IG2N4UEJUR2K",
    "aggregation": {"count":"mean"}
}'

Refresh your data

DATA='{"key": "AMJ1IG2N4UEJUR2K"}'

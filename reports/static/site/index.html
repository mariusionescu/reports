Push data as time series

DATA='{
    "key": "AMJ1IG2N4UEJUR2K",
    "rows": [
        {"animals": 30}
    ],
    "timestamp": 1427963205
}'
curl -XPUT --data "$DATA" http://reports.appixio.com/api/v1/report/1/

Push data as table series

DATA='{
    "key": "AMJ1IG2N4UEJUR2K",
    "rows": [
        {"animal": "bear", "count": 30},
        {"animal": "cows", "count": 40}
    ],
    "index": "animal"
    "timestamp": 1427963205
}'
curl -XPUT --data "$DATA" http://reports.appixio.com/api/v1/report/1/

Visualize data

DATA='{
    "key": "AMJ1IG2N4UEJUR2K",
    "aggregation": {"count":"sum"},
    "start_date": 1427445305,
    "end_data": 1427963705
}'
curl -XPOST --data "$DATA" http://reports.appixio.com/api/v1/report/1/

Refresh your data

DATA='{"key": "AMJ1IG2N4UEJUR2K"}'
curl -XDELETE --data "$DATA" http://reports.appixio.com/api/v1/report/1/
